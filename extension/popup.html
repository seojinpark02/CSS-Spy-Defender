CSS Spy Defender

v1.0.0
Protects against CSS-only fingerprinting attacks. Prevents attackers from collecting browser and device information using only CSS.
Defense Options
Force-loads all conditional URLs in CSS to prevent information leakage through network request patterns.
Disables stylesheets containing trigonometric calc() and env() functions.
May break some websites.
Neutralizes all @container query rules.
May break responsive layouts.
Based on "Cascading Spy Sheets" (NDSS 2025)
